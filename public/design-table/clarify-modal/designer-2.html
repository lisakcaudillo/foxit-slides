Designer 2 — Flow notes
  • Prompt-preview bridge. The user's original prompt sits at the top as a soft quote, and a live preview below the questions reassembles it sentence-style as they answer. Filled tokens animate from grey-italic-placeholder to violet-pill. This is the signature flow move — the user never wonders “what is this thing doing with my prompt?”
  • Conversational header, not interrogation. “Let’s make this sharper” + “takes about 15 seconds” sets expectation and removes the gatekeeping feel. The eyebrow chip (“A few quick details”) softens further.
  • Skip is shameless and dual-exposed. Visible text button bottom-left (“Skip — just generate”) + close X top-right + Esc shortcut. Three escape hatches. None labeled as a downgrade.
  • Optional question is labeled optional. Inline grey “(optional)” on Q3 so users know they can leave it — cuts perceived burden in half.
  • Tab order is linear & HIG-clean. Close → audience chips → depth chips → text input → Skip → Continue. All targets ≥ 44px. Violet focus rings on every interactive element. prefers-reduced-motion kills the rise-in + preview-dot pulse.
  • Keyboard hints visible. Esc Skip / ↵ Continue sit next to the CTA so power users move without reaching for the mouse.
