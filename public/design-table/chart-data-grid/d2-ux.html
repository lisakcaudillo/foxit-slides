Chart block selected

Select a chart block → the inspector reveals the data editor (progressive disclosure, P-UX2). Selecting any non-chart block hides it.

Keyboard nav. Tab/↵ move to next cell, ↑↓←→ jump between cells, ↵ on last cell adds nothing — the grid never traps you. Edits apply on blur/Enter → chart re-renders live.
Paste from Excel. Click the cell area and Ctrl+V, or use this button. First row = headers, first column = labels, rest = values. A toast reports coerced cells.
Delete is reversible & confirmed. Hover a row/column → trash affordance. Click → inline confirm popover (not a modal). Last row/column can't be deleted.
