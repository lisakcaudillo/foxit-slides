Shape · type · spacing

  • Radius: chips 8px, controls 10px, CTA/rows 12px, card 16px. No pills.
  • Type: Inter — title 14.5px/600, brief 12.5px/500 muted, chips 12px, body min 13px.
  • Grid: 8px scale; row = 28 / 28 / 1fr / auto columns (handle · number · body · actions).

Interactive states

  • Default — handle & trailing actions hidden until row hover/focus.
  • Hover — row slate-50 wash; handle + duplicate/remove fade in (slide 3 demo).
  • Active/edit — title input gets 2px violet focus ring + white bg.
  • Layout chip hover — violet border + wash, glyph turns violet (swap affordance).
  • Dragging — lifted card shadow + dashed violet outline; violet drop-line marks target.
  • Generating — per-slide number swaps to ✓ done / ⟳ active / ○ idle; top progress fill; controls locked.

Accessibility

  • All targets ≥ 44px (rows, CTA 48px, icon hit-area padded).
  • Visible violet focus ring (3px / .28 alpha) on every control.
  • Image toggle uses aria-pressed + text label — never color alone.
  • Status conveyed by icon + text ("2 of 5 done"), not color alone. WCAG AA contrast throughout.
  • Plain language: "Generate deck", "Add slide", "Add image", "Split image".
