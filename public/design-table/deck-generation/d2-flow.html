D2 · Flow notes

  • Resting calm. Before you touch anything, the 5 controls are one quiet row of value-pills — icon + current value, no options exposed. Defaults read muted; a changed control turns violet. It reads like a sentence you can amend, not a form.
  • Reveal on intent. Click a pill → a single focused popover opens anchored to it, showing only that control. Audience / Tone / Voice get instant type-ahead; below the presets, “Use ‘…’” turns whatever you typed into the custom value. Detail is a 3-stop segmented control. One popover at a time.
  • Fewest clicks. Type a sentence, hit Generate — 1 action. Tweak one thing first — 3 (pill → choice → Generate). The card never reflows while you configure.
  • Keyboard-first. Tab accepts the ghost example · ⌘K opens the first un-set control · arrows move in a popover, type-to-filter, Enter commits · Esc closes · ⌘↵ generates. Visible focus rings throughout; prefers-reduced-motion fully honored.
  • The Generate moment. Pills + actions cross-fade out, an accent orb spins up, the status writes itself, and a deck silhouette assembles — a premium hand-off, not a spinner.
