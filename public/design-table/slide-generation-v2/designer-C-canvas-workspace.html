ROUND 2 Designer C — “The Easel” · Spatial creative workspace

Round 1 built a form (prompt + pill row) and Lisa rejected it as unoriginal. This rebuilds the surface as a physical workbench: the deck-to-be is already on the easel as ghosted, drifting preview slide-cards casting warm shadows; the prompt is a brief pinned with a brass clip; the controls are an artist's tool tray bolted to the right rail — a brass slide-count dial that physically adds/removes cards, a chunky physical toggle switch for Auto art, paint-swatch themes, a tool drawer that slides open, and a brass START plunger to generate. Color is one deliberate palette: violet = the studio's house ink, Foxit orange = “armed/live” (only the auto-art switch when ON + the plunger), linen = canvas/wood. The toggle reads by knob position + label + glyph, never color alone. Motion (drift, parallax tilt, spring knob, card add/remove) fully disables under prefers-reduced-motion.
