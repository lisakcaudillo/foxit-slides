Designer A — Hero Studio. The page is an artwork you write on, not a form you fill in.

  • Original (not Round 1): R1 calmed clutter by bleaching it to neutral pills + one accent. This makes the surface a living aurora canvas you compose over — Procreate, not Settings.
  • Hero concept: drifting blurred color-blobs + frosted veil (Compose's own glassmorphism / CoverArt material), prompt typed directly over the painting.
  • Rich yet cohesive: every hue samples ONE kit ramp (pink→lavender→sky / magenta→blue). Vivid but monochromatic-in-spirit — R1's problem was incoherent color, not color.
  • Tactile toggle: the auto-image enabler is the one true on/off, so it's a real chunky rocker — recessed well, travelling knob, press depth. State = position + OFF/ON word + tick, never color alone.
  • Motion is a material: aurora drift, grain sweep, hover-lift, press-depth, spring knob travel, CTA sheen.
  • A11y: ≥16px text, ≥44px hits, focus rings on every object, role="switch" toggle; prefers-reduced-motion freezes the whole painting.
