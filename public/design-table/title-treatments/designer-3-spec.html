Designer 3 — Title Treatment Spec Library

A family of six cover treatments that each draw their identity from a different dimension of the theme token set — so two decks never read as the same template recolored. Every part of every cover traces back to a real Theme field (titleFont, titleColor, bodyColor, pageBg, pagePattern, plus the per-theme CoverArt motif). This sheet extends the existing cover-tiers.ts system: treatments map onto type / split / photo tiers so the live image pipeline, scrim and contrast logic keep working unchanged. Renderings below are live, at 16:9.
