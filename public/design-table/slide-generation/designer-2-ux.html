UX Flow lens — rationale

  • Diagnosis: the clutter is too many controls exposed at once, not just too many colors. Five equal-weight controls = "five decisions before I can start." Fix = reduce simultaneous exposure, then one accent survives.
  • Primary Prompt + Generate — the only two things that glow. Generate is the single gradient CTA on the page (violet = AI).
  • Secondary Slides count + Theme — always visible on the card foot rail, but monochrome slate, no accent badges. Theme's identity = one small swatch dot, not a rainbow.
  • Advanced Upload, Auto-images, Customize collapse behind ONE "Options" disclosure (P-UX2). Most users type → Generate and never open it (P-UX10 Calm Default).
  • Toggle: Auto-images is a real on/off switch (Lisa's model), placed inside Options because it's an opt-in cost-bearing behavior — with an inline credit note so cost is honest at the decision point. State shown by label + knob + check, never color alone (HIG).
  • A11y: single semantic accent; switch is role="switch" with 44px hit area; prefers-reduced-motion kills all motion; visible focus rings; disclosure keyboard-reachable.
