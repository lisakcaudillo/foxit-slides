Designer 1 · Visual Design lens
Color strategy: one-accent system. Every secondary control (Upload, Slides, Customize, Theme, auto-image) drops to a single neutral ghost-glass skin — slate ink on translucent white, no colored icon badges. This kills the four-way color fight (blue / neutral / violet / tri-color gradient) that made the row feel cluttered.
The only saturated color on the page is the violet AI accent, reserved for the Generate CTA (kit gradient) — so the eye lands on the one thing that matters. Toggle form: a real iOS-style switch with an inline label — OFF is calm slate (P-UX10 Calm Default), ON slides the knob right + turns the track violet + shows the word "On" (state by position first, color second — colorblind-safe). A11y: control text slate-700 ≥ 4.5:1 on glass; CTA text violet-950 on the kit gradient (WCAG AA per CLAUDE.md); switch is role=switch with a focus ring; hit-areas ≥ 44px.
