Designer 3 · UI Spec / Component — clutter fix via one disciplined control family

  • One primitive, five instances. Upload / Slides / Auto-images / Customize / Theme all become the same .ctl pill — 32px tall, 10px radius, glass surface, slate-700 / 600 label. No bespoke colors.
  • Colour = state, not decoration. Secondary controls are neutral at rest; the single violet accent appears only on the CTA and on an active control (open / ON / chosen).
  • The switch. Auto-images is now a real enabler: 36×20 track + sliding thumb, OFF (slate) → ON (violet, thumb right, "On" text). Shown both ways below.
  • Icons monochrome. The colored badge squares are deleted (a main clutter source); Lucide icons sit at slate-500, inheriting accent only when active.
  • A11y. 44px tap targets, slate-700 text ≥4.5:1, 2px violet focus ring everywhere, prefers-reduced-motion honoured.
